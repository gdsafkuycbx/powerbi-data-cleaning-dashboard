# powerbi-data-cleaning-dashboard
This project focuses on cleaning raw data and building an interactive dashboard using Power BI.

 Data Cleaning (Power Query)
1.Removed duplicates
2.Handled missing values
3.Validated emails and phone numbers
4.Formatted date and numeric fields
 Dashboard Features
1.Total Customers KPI
2.Total Revenue KPI
3.Average Purchase KPI
4.Sales by City (Bar Chart)
5.Sales Trend (Line Chart)
6.Filters (City, Date)
Key Insights
1.Identified top-performing cities
2.Observed sales trends over time
3.Detected invalid/missing data
 Tools Used
1.Power BI
2.Excel
3.Power Query
3.DAX
